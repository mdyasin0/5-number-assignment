Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an element in the DOM:
1️. getElementById It's used when the element has an ID.
2️. getElementsByClassName It's used when the element has a class, and it selects all elements with the same class.
3️. getElementsByTagName It's used when selecting by tag name like p, h1 , h5, div, etc.
4️. querySelector()Selects the first matching element using a CSS selector.
5️. querySelectorAll() Selects all matching elements using a CSS selector.

Question-3: What is event delegation in the DOM?

Event delegation means adding an event listener to the parent element, so there's no need to add events to each child element. It has some benefits:
1.Event handling with less code.
2.Performance improves.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where the event starts from the element where it was triggered and goes up step by step to the document.
